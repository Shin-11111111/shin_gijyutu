Section 01

1 HTMLとは何か？

HTML（HyperText Markup Language）は、 ウェブページの「骨格・構造」を作るための言語です。 あなたが毎日見ているYouTubeやGoogleのページも、すべてHTMLで作られています。

HTMLはマークアップ言語と呼ばれます。 「マークアップ」とは、文章の一部に目印（タグ）をつけて意味を伝えること。 「ここは見出し」「ここは画像」「ここはリンク」……のように、 ブラウザへ指示を出す言語です。

📄

テキストで書ける

メモ帳などのテキストエディタで書けます。特別なソフトは不要です。

🌐

ブラウザが読む

Chrome・Safari・Edgeなどのブラウザがファイルを読んで画面に表示します。

🔖

タグで意味をつける

「<>」と「</>」で文章を囲んで「これは何か」を伝えます。

⚡

すぐ確認できる

ファイルを保存→ブラウザで開く、たったこれだけで結果が確認できます。

Section 03

3 タグのしくみ

HTMLの最大の特徴は「タグ」です。 タグは <開始タグ>内容</終了タグ> の形で書きます。

タグの基本パターン 
<!-- 開始タグ <h1> と 終了タグ </h1> で文章を囲む -->
<h1>これは大見出しです</h1>

<!-- 段落(パラグラフ) -->
<p>これは本文の段落です。</p>

<!-- リンク(属性 href でURLを指定) -->
<a href="https://www.google.com">Googleへ</a>

<!-- 画像(終了タグなし・src で画像ファイルを指定) -->
<img src="neko.jpg" alt="ねこの写真">

よく使うタグを表で確認しましょう：

タグ 意味・用途 書き方の例
<h1>〜<h6> 見出し（h1が最大・h6が最小） <h1>大きな見出し</h1>
<p> 段落・本文テキスト <p>本文テキスト</p>
<a> リンク（クリックで移動） <a href="URL">テキスト</a>
<img> 画像の表示 <img src="画像ファイル">
<ul> / <li> 箇条書きリスト <ul><li>項目</li></ul>
<table> 表（テーブル） <table>〜</table>
<div> まとめる箱（グループ化） <div>グループ</div>
<style> CSS（デザイン）を書く場所 <style> h1{color:red;} </style>
Section 04

4 最初のHTMLを書いてみよう

HTMLファイルには決まった「お作法」があります。 下のコードが基本テンプレートです。まずはこれをメモ帳に貼り付けてみましょう。

index.html — 基本テンプレート 
<!-- 1行目: HTMLファイルですよ、という宣言 -->
<!DOCTYPE html>

<!-- html タグ: 全体を囲む。lang="ja"は日本語を意味する -->
<html lang="ja">

  <!-- head: ブラウザへの設定（画面には表示されない） -->
  <head>
    <meta charset="UTF-8">  <!-- 文字コード（日本語対応） -->
    <title>はじめてのHTML</title>  <!-- タブに表示されるタイトル -->
  </head>

  <!-- body: 実際に画面に表示する内容 -->
  <body>
    <h1>こんにちは、世界！</h1>
    <p>はじめてのHTMLファイルです。</p>
  </body>

</html>
ブラウザ表示イメージ

こんにちは、世界！

はじめてのHTMLファイルです。

次に、CSSで色をつけてみましょう：

CSSで見た目を変える（<style>タグを追加） 
<head>
  <meta charset="UTF-8">
  <title>スタイルを加えよう</title>

  <!-- ↓ ここにCSSを書く -->
  <style>
    body {
      background-color: #f0f8ff;  /* 背景色 */
      font-family: sans-serif;
    }
    h1 {
      color: #e74c3c;   /* 文字色（赤） */
      text-align: center;
    }
    p {
      font-size: 18px;
      color: #2c3e50;
    }
  </style>
</head>
ブラウザ表示イメージ（CSSあり）

こんにちは、世界！

はじめてのHTMLファイルです。背景色と文字色が変わりました。

さらに、JavaはHTMLとは違う「本格的なプログラミング言語」です。比べてみましょう：

Hello.java — Javaの基本プログラム 
// Javaのプログラムはクラスの中に書く
public class Hello {

    // main メソッド: プログラムの開始地点
    public static void main(String[] args) {

        // 画面に文字を表示する
        System.out.println("こんにちは、世界！");

        // 変数を使った計算
        int score = 100;
        System.out.println("得点: " + score);

        // くり返し（for文）
        for (int i = 1; i <= 3; i++) {
            System.out.println(i + "回目のループ");
        }
    }
}
スクロールして実行しよう！
Hello.java ↓ 下にスクロールすると実行されます
▶ このエリアが画面に入るとプログラムが走ります

🆚 HTMLとJavaの違い

比較項目 HTML Java
種類 マークアップ言語 プログラミング言語
主な用途 ウェブページの構造 アプリ・システム開発
実行方法 ブラウザで開く コンパイルして実行
計算・くり返し できない できる（for文・if文）
使われる場所 ウェブサイト Android・業務システムなど

📋 ファイルの作り方（手順）

1

メモ帳（テキストエディタ）を開く

Windowsなら「メモ帳」、Macなら「テキストエディット」で新規ファイルを作成します。

2

HTMLコードを書いて保存する

ファイル名を「index.html」にして保存します。拡張子が .html であることが重要です。

3

ブラウザで開く

保存したファイルをダブルクリックするか、ブラウザにドラッグ＆ドロップします。

4

修正 → 保存 → 再読み込みを繰り返す

コードを変えたら保存して、ブラウザの「更新（F5）」を押します。これがプログラミングの基本サイクル！

🧩 理解度チェック

Q1. HTMLの正式名称は何ですか？

✅ HyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）
「ウェブ上の文書に意味をつける言語」という意味です。

Q2. HTMLのタグはどんな記号で書きますか？

✅ 山括弧（< と >）を使います。例：<h1>見出し</h1>
開始タグと終了タグ（/が入る）のペアで書きます。

Q3. HTMLとCSSの違いは何ですか？

✅ HTMLは「構造・骨格（何を置くか）」を作り、CSSは「見た目・デザイン（どう見せるか）」を担当します。

Q4. HTMLファイルを確認するには何を使いますか？

✅ ウェブブラウザ（ChromeやEdgeなど）を使います。ファイルをダブルクリックして開きます。

Q5. 見出しを表すタグは何ですか？一番大きいものは？

✅ 見出しタグは <h1> 〜 <h6> です。一番大きい（重要な）見出しは <h1> です。

Q6. <a href="..."> タグは何のために使いますか？

✅ リンク（ハイパーリンク）を作るためのタグです。クリックすると href="" に書いたURLへ移動します。
