プログラムって何？ 〜順次実行〜

命令を上から順番に実行する
プログラムは「やること」を順番に書いた指示書です。コンピューターは上から下へ、一行ずつ正確に実行します。
① 起きる
→
② 歯を磨く
→
③ 服を着る
→
④ 学校へ行く
コンピューターはこの順番を絶対に守ります。③を飛ばして④はできません。これを順次実行といいます。
 ステップ実行を体験しよう
「次へ」を押すと、プログラムが一行ずつ実行される様子が見えます。
name = "さくら" age = 14 print("こんにちは、" + name) print("あなたは" + age + "歳です")
1
変数 name に "さくら" を入れる
2
変数 age に 14 を入れる
3
"こんにちは、さくら" を画面に表示
→ こんにちは、さくら
4
"あなたは14歳です" を画面に表示
→ あなたは14歳です

条件：score ≥ 80

繰り返し 〜同じ処理を何度でも〜

for ループと while ループ
同じ処理を繰り返す仕組みです。「何回」か決まっているなら for、「条件が続く限り」なら while を使います。
for i in range(5):     print("* " + i)
 繰り返し回数を変えよう
5
hp = 10
while hp > 0:
    print("HP: " + hp)
    hp = hp - 3
print("ゲームオーバー")
 ダメージ量を変えよう（RPGのHP）
3
while ループは条件が False になるまで永遠に続きます。条件が絶対に False にならないと「無限ループ」になるので注意！

関数 〜処理をまとめて再利用〜

関数 ＝ 「レシピ」のようなもの
関数とは、よく使う処理を「ひとかたまり」にして名前をつけたものです。一度作れば何度でも呼び出せます。
引数（入力）
name = "たろう"
→
Function
greet()
→
戻り値（出力）
"こんにちは、たろう！"
# 関数を定義する
def greet(name):
    message = "こんにちは、" + name + "！"
    return message

def judge(score):
    if score >= 90: return "S"
    elif score >= 80: return "A"
    elif score >= 70: return "B"
    elif score >= 60: return "C"
    else: return "D"

# 関数を呼び出す
print(greet("たろう"))
print(greet("はな"))
 関数に渡す値を変えよう
85
greet() + judge() の実行結果
関数を使うと同じコードを何度も書かなくて済みます。これをDRY原則（Don't Repeat Yourself）といいます。
